Collects and stores multiple data points based on the selected generation timing. 

This method generates and stores {\ttfamily m\+\_\+num\+Of\+Data\+Points} data points into {\ttfamily m\+\_\+physical\+Data} using the timing specified by {\ttfamily m\+\_\+generation\+Timing}\+:
\begin{DoxyItemize}
\item {\bfseries{e\+Immediate}}\+: Generates all data points as quickly as possible without delay.
\item {\bfseries{e\+Periodic}}\+: Generates data points with a fixed delay ({\ttfamily m\+\_\+period\+For\+Generation}) between each point.
\item {\bfseries{e\+Async}}\+: Generates data points with a random delay between 100 and 300 milliseconds.
\end{DoxyItemize}
