void Data\+Processor\+::moving\+Average\+Filter (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Applies a moving average filter to the raw data. 

This function performs a simple moving average filtering operation on the raw data ({\ttfamily m\+\_\+raw\+Data}), smoothing the data by averaging each point with its neighboring points. The window size of the filter is defined by {\ttfamily m\+\_\+window\+Size}. The filtered result is stored in {\ttfamily m\+\_\+processed\+Data}.

The algorithm works as follows\+:
\begin{DoxyEnumerate}
\item The raw data is padded with the first and last values to handle edge cases.
\item For each data point, the function calculates the sum of the current point and its neighbors within the window size, then scales the sum to calculate the average.
\item The averages are stored in the {\ttfamily m\+\_\+processed\+Data} vector. 
\end{DoxyEnumerate}\Hypertarget{class_data_processor_af87d3a247b88de5866419c89d533a9c7}\index{DataProcessor@{DataProcessor}!setRawData@{setRawData}}
